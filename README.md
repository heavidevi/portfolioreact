## ğŸš€ Deployment

- **URL:** [https://portfolioreact-wheat.vercel.app/](https://portfolioreact-wheat.vercel.app/)

```bash

# Deploy to Vercel

```

The `vercel.json` file is configured for SPA routing:
```json
{
  "rewrites": [
    {
      "source": "/(.*)",
      "destination": "/index.html"
    }
  ]
}
```
